# 高阶函数

一个函数接收另一个函数作为参数或者返回值为一个函数，这种函数就称之为高阶函数。

数组中的高阶函数：

  1. Array.prototype.map
  2. Array.prototype.reduce
  3. Array.prototype.filter
  4. Array.prototype.sort
