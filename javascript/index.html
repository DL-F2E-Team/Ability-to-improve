# 闭包

  1. 可以访问其他函数内变量的函数，叫做闭包。或者说在一个函数内部可访问该函数内部局部变量的函数，作用就是让函数外部可以访问函数内部局部变量。
  2. 闭包可以用来保存一个需要持久保存的变量，可以模拟命名空间。
  3. 闭包在IE种会造成内存泄露（IE BUG）
    function foo () {
        var ex = 1;
        function bar () {
            ex++
            return ex
        }
        return bar
}
var func = foo()
