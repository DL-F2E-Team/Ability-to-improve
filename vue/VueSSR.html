# Vue SSR

  • 利于SEO搜索引擎优化
  • 可以被爬虫抓取
  • SSR直接将HTML字符串传递给游览器，大大加快首屏加载时间
  • SSR占用更多的内存和CPU资源
  • 一些常用的游览器API可能无法正常使用
  • 在Vue中只支持 beforeCreate 和 created 两个生命周期
